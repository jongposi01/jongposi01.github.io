개인정보처리방침

시행일: 2026년 5월 19일

1. 개인정보처리방침 안내

티지앰지(TJ&MJ)는 「가벼운 사업자조회」 앱을 운영함에 있어 이용자의 개인정보를 중요하게 생각하며, 관련 법령을 준수하기 위해 본 개인정보처리방침을 공개합니다.

2. 수집하는 개인정보 항목

「가벼운 사업자조회」 앱은 회원가입 기능을 제공하지 않으며, 이용자의 이름, 연락처, 이메일, 주소 등 개인을 식별할 수 있는 개인정보를 직접 수집하지 않습니다.

다만, 사업자등록번호 진위 확인 및 상태 조회 기능 이용 시 사용자가 입력한 사업자등록번호, 개업일자, 대표자명 등의 정보가 조회 요청을 위해 공공데이터 API로 전송될 수 있습니다.

3. 개인정보의 이용 목적

앱에서 입력되는 정보는 다음 목적을 위해 사용됩니다.

  • 사업자등록번호 진위 여부 확인
  • 사업자 상태 조회
  • 공공데이터 API 요청 및 결과 표시

입력된 정보는 앱 내에서 별도로 저장하거나 다른 목적으로 이용하지 않습니다.

4. 개인정보의 보관 및 파기

「가벼운 사업자조회」 앱은 사용자가 입력한 조회 정보를 별도 서버에 저장하지 않습니다. 조회 요청 완료 후 앱 내에 지속적으로 보관하지 않으며, 앱 종료 또는 화면 초기화 시 정보가 삭제될 수 있습니다.

5. 제3자 제공 및 외부 API 이용

본 앱은 사업자등록번호 진위 확인 및 상태 조회를 위해 공공데이터포털에서 제공하는 국세청 사업자등록정보 진위확인 및 상태조회 API를 이용할 수 있습니다.

사용자가 입력한 조회 정보는 해당 API 요청 처리를 위해 공공데이터 제공기관으로 전송될 수 있습니다.

데이터 출처: 공공데이터포털 / 국세청 사업자등록정보 진위확인 및 상태조회 서비스

6. 광고 서비스

본 앱은 서비스 운영을 위해 Google AdMob 광고를 포함할 수 있습니다. 광고 제공 과정에서 Google 및 광고 네트워크가 광고 식별자, 기기 정보, 앱 이용 정보 등을 수집할 수 있습니다.

Google의 개인정보 처리에 관한 자세한 내용은 Google 개인정보처리방침을 참고하시기 바랍니다.

7. 권한 사용 안내

「가벼운 사업자조회」 앱은 기본 조회 기능 제공을 위해 필수적인 최소 권한만 사용합니다. 별도의 위치 권한, 연락처 권한, 카메라 권한 등 민감 권한은 사용하지 않습니다.

8. 비공식 앱 안내

「가벼운 사업자조회」는 국세청, 공공데이터포털 또는 정부기관이 직접 운영하는 공식 앱이 아닙니다. 공공데이터 API를 활용하여 사업자등록번호 조회 기능을 제공하는 비공식 앱입니다.
